# chrome browsing tracker

a Chrome extension that tracks your own browsing activities to provide data for behavior changes.

## permissions

This extension stores your browsing activities only locally.

- history
- tabs
- storage
- unlimitedStorage

## development

```bash
npm install
npm start
```
